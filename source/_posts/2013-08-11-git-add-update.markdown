<!--more-->

> $ git add -u
> 
> warning: The behavior of 'git add --update (or -u)' with no path argument from a
subdirectory of the tree will change in Git 2.0 and should not be used anymore.
To add content for the whole tree, run:
>  git add --update :/
>  
>  (or git add -u :/)
